# 기술 중간평가

---

# React
<br />

리액트를 써본적이 없는데 전체적인 흐름을 어느정도 깨우침

classNames를 통한 스타일제어 등이 편리함

최적화를 정말 잘하려면 딥하게 공부하는 시간이 많이 필요함

Component를 가져다 사용하는게 정말 용이함

Hooks 사용법을 배우고 활용함

---

# TailwindCSS


처음에 몇일 고생했지만 엄청 편함

Bootstrap 같은 느낌을 받음

생각보다 어렵지 않고 사용성이 좋음

---

# Prisma

러닝커브가 있고 RDB였다면 어떤 SQL이 날아갈지 예측이 안됨

JPA 강의를 배웠었는데 best pratice가 강제되어 잘 녹아있음

기본적인 CRUD만 진행했지만 편리함

데이처 요청관련해서 JS, TS로 작성할 수 있고 어떤 데이터를 요청할 수 있는지 알 수 있어서 개발속도도 빠름

기본적으로 제공하는 Database Studio가 생각보다 잘되어있음

---

# NextJS

디버깅 하는게 너무 불편함

파일 구조로 핸들러를 매핑하는게 익숙하지가 않음

어느정도 React, JS에 대한 이해도를 요구함
