여러분 이렇게 공용 컴포넌트를 만들었습니다!?

```tsx
import classNames from 'classnames';

export default function Button({
  text,
  icon = null,
  type = 'primary',
  size = 'middle',
  outline = false,
  dashed = false,
  disabled = false,
  customCSS = '',
  onClick,
}) {
  return (
    <button onClick={onClick}>
      {버튼을 만들어내는 아름다운 로직...}
    </button>
  );
}

```

---

이제부터 버튼은 모두가 공용으로 사용합니다.

```tsx
<Button text="버튼" type="primary" />
```
